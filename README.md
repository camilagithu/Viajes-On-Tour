## Viajes On Tour

**Viajes On Tour** es un sitio web tipo **One Page** diseñado como proyecto académico para la asignatura **Fullstack II**.  
Su objetivo es apoyar la gestión de giras de estudio mediante una interfaz clara, responsiva y validada con JavaScript.

## Características principales

- Menú de navegación fijo con scroll suave entre secciones.  
- Secciones incluidas: Inicio, Giras, Nosotros, Blog y Contacto.  
- Cards de información para giras y artículos.  
- Formulario validado en JavaScript (nombre, correo con dominios permitidos, comentario ≤ 500).  
- Diseño responsive con CSS puro.
